### 简介

基于[element-plus-nuxt-starter](https://github.com/element-plus/element-plus-nuxt-starter)改造开发。

由Vue3 + Nuxt + Element PLUS实现的客户端展示页面，仅供线上简历展示使用。

推荐使用node20以上的开发环境，否则pnpm安装可能会导致一些问题。

### 安装依赖

```bash
pnpm install
```

### 开发

启动开发环境打开地址 `http://localhost:3000`

```bash
pnpm dev
```

### 生产

```bash
pnpm build
```
